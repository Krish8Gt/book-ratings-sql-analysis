# Book Ratings & User Behavior Analysis

## Overview
End-to-end SQL analytics project analyzing user engagement and book ratings.

## Dataset
- 2,78,858 users
- 2,70,928 books
- 10,31,582 ratings

## Key Work
- Data cleaning and normalization
- ISBN standardization
- Relational modeling with PK/FK constraints
- Country, age, popularity, and historical analysis

## Tools
- MySQL
- GitHub Desktop
- SQL
